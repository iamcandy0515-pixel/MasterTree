# [작업계획서] 사용자 앱 세션 관리 및 자동 로그인 구현 (V1.1)

## 1. 개요 및 목적
`DEVELOPMENT_RULES.md` 가이드라인에 따라 사용자 앱(`flutter_user_app`)의 세션 안정성을 확보하고, 불필요한 재로그인을 방지하기 위한 자동 로그인(Auto-Login) 기능을 구현합니다.

## 2. 준수 사항 (Compliance)
- **버전 고정**: Flutter `3.7.12` / Dart `2.19.6` 환경을 엄격히 준수합니다.
- **200줄 제한 원칙**: 파일 분리 시 단일 파일 소스가 200줄을 초과하지 않도록 모듈화합니다.
- **소스 정합성**: 모든 수정은 `diff` 기반으로 진행하며, 기존 로직 유실 방지를 위해 수정 전/후 정합성을 체크합니다.
- **린트 체크**: 작업 완료 후 `flutter analyze`를 통해 스타일 및 문법 오류가 없음을 증명합니다.

## 3. 현황 분석
- **문제점**: `main.dart`가 `LoginScreen`을 직접 호출하여 로컬에 저장된 Supabase 세션을 활용하지 못함.
- **로직 결함**: `AuthViewModel`에서 세션 복구 후 유저의 승인 상태(`status`)를 서버에서 재확인하는 과정이 누락됨.
