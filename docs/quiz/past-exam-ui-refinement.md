- **목적**: '기출문제 상세' 화면의 시각적 완성도를 높이고 특정 텍스트의 가독성을 개선함.
- **작업 범위**:

    2.  **텍스트 처리**: 특정 단어('양단', '면적') 사이에 줄바꿈 특수문자가 포함된 경우 이를 제거하여 한 줄로 표시되도록 처리.

## 3. 사후 점검 (Review)

- **완료된 결과 (Result)**:
    - `PastExamDetailScreen`의 모든 카드형 요소에서 테두리(`border`)를 제거하여 배경색 위주의 깔끔한 프레임리스 디자인을 적용했습니다.
    - `_formatText` 유틸리티 기능을 통해 문제, 해설, 보기 텍스트 내의 '양단[줄바꿈]면적' 패턴을 감지하여 자동으로 한 줄('양단 면적')로 합치는 보정 로직을 구현했습니다.
- **향후 문제점 및 리스크 분석 (Risk Analysis)**:
    - **가독성**: 테두리가 제거되었으므로 인접한 섹션 간의 구분이 모호해 보일 수 있습니다. 현재 각 섹션 사이에 충분한 `SizedBox` 여백을 두어 가독성을 유지하도록 설계했습니다.
    - **정규식 매칭**: 현재 정규식은 '양단'과 '면적' 사이의 공백과 줄바꿈만 처리합니다. 다른 단어 조합에서도 유사한 이슈가 발생할 경우 해당 리스트에 추가 정의가 필요합니다.
