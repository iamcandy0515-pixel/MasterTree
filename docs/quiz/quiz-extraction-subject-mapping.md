# '산림필답' 등 과목명 동적 파싱 정규화(Normalization) 구현

## 1. 목적 및 원인 파악 (Plan)

- **증상:** 구글 드라이브나 로컬에서 전달받은 PDF 파일 이름(`산림필답_2013년_1회.pdf` 등)을 토대로, `QuizExtractionStep2ViewModel`에서 과목명을 자동 추출하는데 `parts[0]` 처리 로직에 의해 정규 규격과 다른 `산림필답` 같은 텍스트가 그대로 드롭다운 `_extractedSubject`의 값으로 입력되는 문제가 있었습니다.
- **조치 방향:** 파일 이름 또는 1차적으로 나눠진 접두어(`산림필답` 등)를 검사하여 앱에서 지원하는 **표준 6대 과목명(산림기사, 산림산업기사, 조경기사, 조경산업기사, 산업안전기사, 산업안전산업기사)**으로 치환(Mapping)해주는 정규화 로직을 도입해야 합니다.

## 2. 작업 내용 (Execute)

- `quiz_extraction_step2_viewmodel.dart` 분기문 리팩토링:
    - 파일명을 1차적으로 쪼갠 후 최종 값이 결정되는 `_extractedSubject` 변수가 할당된 "직후" 시점에 **정규화 과정(Normalization)**을 추가했습니다.
    - 다음은 적용된 치환 룰셋(Rule-set)입니다:
        - `산림산업기사`, `산림산업필답` 키워드 포함 -> **산림산업기사**
        - `산림기사`, `산림필답` 키워드 포함 -> **산림기사**
        - `산업안전산업기사` 키워드 포함 -> **산업안전산업기사**
        - `산업안전기사` 키워드 포함 -> **산업안전기사**
        - `조경산업기사` 키워드 포함 -> **조경산업기사**
        - `조경기사` 키워드 포함 -> **조경기사**

## 3. 사후 점검 (Review)

- **Result:**
    - 이제 더이상 `산림필답` 이라는 비표준 과목명이 드롭다운에 강제로 추가되는 일 없이, 똑똑하게 **산림기사**로 매핑되어 화면에 출력됩니다.
- **Risk Analysis:**
    - `contains` 기반 탐색이므로, 만약 `산림산업기사필답` 같은 파일명이라면 '산림산업기사' 키워드가 우선 감지되어 정상 매핑됩니다. 만약 전혀 알 수 없는 `일반기계기사_2020` 같은 파일명이 오더라도 이전 단계에서 개발한 동적 Item 추가 방어 코드 덕분에 앱 크래시가 나지 않고 그대로 보여준 뒤 유저가 직접 드롭다운 리스트를 통해 선택할 수 있습니다.
