# 🌲 수목 힌트 배치 업데이트 가이드

## 📋 개요

10개 수목 단위로 대표힌트, 잎힌트, 수피힌트, 겨울눈힌트를 일괄 업데이트하는 템플릿입니다.

---

## 📁 파일

- **템플릿:** `scripts/data/tree_hints_batch_template.json`
- **업데이트 스크립트:** `scripts/update_tree_hints_from_template.ts`

---

## 🚀 사용 방법

### **1단계: Tree ID 확인**

업데이트할 수목의 ID를 확인합니다.

```bash
npx ts-node scripts/generate_main_hints_report.ts
```

또는 Supabase에서:

```sql
SELECT id, name_kr, category FROM trees
WHERE name_kr IN ('마가목', '매자나무', '음나무', ...)
ORDER BY name_kr;
```

---

### **3단계: 업데이트 실행**

```bash
cd nodejs_admin_api
npx ts-node scripts/update_tree_hints_from_template.ts
```

**⚠️ 주의:**

- 템플릿 파일명이 `tree_hints_batch_template.json`이면 스크립트를 수정해야 합니다.
- 또는 `tree_hints_template.json`으로 복사하여 사용하세요.

```bash
# 방법 1: 파일 복사
cp scripts/data/tree_hints_batch_template.json scripts/data/tree_hints_template.json

# 방법 2: 업데이트 실행
npx ts-node scripts/update_tree_hints_from_template.ts
```

## ⚠️ 주의사항

1. **Tree ID 확인 필수**: 잘못된 ID는 업데이트 실패의 원인이 됩니다.
2. **선택적 입력**: 모든 힌트를 입력할 필요는 없습니다. 필요한 것만 입력하세요.
3. **빈 항목 제거**: `tree_id: 0`이거나 `수목명: ""`인 항목은 자동으로 스킵됩니다.
4. **백업**: 중요한 데이터이므로 작업 전 데이터베이스 백업을 권장합니다.
