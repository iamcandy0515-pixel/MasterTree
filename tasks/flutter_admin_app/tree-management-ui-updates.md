# 작업 계획서: 수목 관리 UI/UX 개선 (V6)

## 1. 개요

관리자 앱의 수목 관리 및 등록 화면에서 레이아웃 배치, 입력 항목 조정, 네비게이션 로직 및 디자인 요소(스마트 태그, 버튼 스타일, 미리보기 뒤로가기)를 개선하여 사용자 운영 효율성을 극대화합니다. 특히 '수목 등록(신규)' 화면에서는 데이터 무결성을 위한 필수 입력 체크와 직관적인 세로 배치 레이아웃을 적용합니다.

## 2. 분석 및 제안: 이미지와 힌트의 카테고리별 연동

사용자님의 요청에 따라 **이미지와 힌트를 카테고리별로 연동**하는 방식을 적용합니다.

- **데이터 무결성 확보**: 특정 이미지(예: 수피)에 대한 힌트가 해당 이미지와 1:1로 매칭되도록 보장하여 입력 오류를 방지합니다.
- **동선 최적화**: 사용자가 특정 카테고리(예: '잎')를 선택하면 해당 이미지와 힌트 입력창이 하나의 그룹으로 활성화되어 입력을 가속화합니다.

## 3. 작업 내용

### 3.1 명칭 및 네비게이션 개선

- **수목 관리 리스트 화면**: '수목현황 일람' -> **'수목관리'** (제목 변경).
- **수목 등록(신규) 화면**:
    - 화면 제목: '새 나무 등록' -> **'수목 등록(신규)'**.
    - 필드 명칭: '한글 이름(필수)' -> **'수목명'**.
    - 버튼 명칭 및 스타일: 상단 '등록' 버튼을 **'TextButton'** 스타일로 변경.

### 3.2 수목 등록(신규) 화면 레이아웃 및 검증 (`add_tree_screen.dart`)

화면의 입력 항목을 사용자의 작업 흐름에 맞춰 **세로로 순차 배치**하고 필수 값을 검증합니다.

1. **기본 정보**: 수목명, 학명, 분류(1차/2차) 섹션을 최상단에 배치 (왼쪽 정렬).
2. **이미지 구분 및 유닛**: '이미지 구분' 콤보박스(카테고리 선택) 배치.
3. **이미지 박스**: 선택된 카테고리의 이미지를 업로드하거나 Ctrl+V로 붙여넣는 영역.
4. **힌트 입력 상자**: 해당 이미지에 대한 힌트를 입력하는 상자 (세로 2줄 크기 확보).
5. **퀴즈 오답**: 최하단에 고정된 2개의 오답 입력란 배치.
6. **필수 입력 체크 (Validation)**:
    - **수목명**: 입력 필수.
    - **이미지 및 힌트**: 각 카테고리별로 이미지와 힌트가 모두 입력되었는지 확인 (둘 중 하나만 누락되어도 저장 불가).

### 3.3 수목 관리 상세 화면 공통 개선 (`tree_detail_screen.dart`)

- **AppBar 저장 버튼**: 'DB저장' `ElevatedButton`을 `TextButton`으로 변경 (NeoColors.acidLime 유지).
- **저장 로직**: 저장 완료 후 화면 이탈 없이 상태 유지 (`Navigator.pop` 제거).
- **기본 정보 배치**: 내용을 **왼쪽 정렬**(`CrossAxisAlignment.start`)로 수정.
- **퀴즈 오답 섹션**: 명칭을 '퀴즈 오답'으로 변경하고 입력란을 **2개로 고정** (오답 추가 버튼 삭제).
- **스마트 태그 시스템**: 오답 영역 하단에 스마트 태그(Chip)를 배치하여 카테고리별 힌트/이미지 입력을 지원.

### 3.4 입력 항목 조정 (`add_tree_screen.dart`)

- 기존 '난이도' 항목을 삭제하고, **'수목 2차 선택'** (상록수, 활엽수) 콤보박스 추가.
