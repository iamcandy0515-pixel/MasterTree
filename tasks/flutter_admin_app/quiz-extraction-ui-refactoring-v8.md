# Task: Quiz Extraction UI Labels, Upsert Logic & Image Cleanup Refactoring

## 1. ANALYSIS (연구 및 분석)

### 1-1. UI 라벨 일관성 확보

- 모든 추출 모듈의 타이틀과 라벨을 더 명확하고 간결한 용어로 통일 완료.

### 1-2. DB 저장 로직 및 이미지 관리 고도화

- **Upsert 강화**: 동일 연도/회차/과목의 동일 문제 번호 존재 시 기존 레코드를 업데이트(`ON CONFLICT DO UPDATE`)하여 중복 방지.
- **이미지 자동 정리**: DB 저장 시 기존 레코드에 있던 이미지와 새로운 데이터의 이미지를 비교.
    - **요구사항**: 화면에서 삭제되거나 변경되어 더 이상 사용되지 않는 이미지는 스토리지(Supabase)에서 즉시 삭제하여 용량 낭비를 방지.
- **연관 데이터**: 확정된 **'유사 기출문제'** ID 목록(`related_quiz_ids`)을 함께 저장.

### 1-3. 힌트(Hint) 데이터 관리

- 단일 JSONB 컬럼 내에서 배열 구조로 여러 힌트를 유연하게 관리.

## 3. SOLUTIONING (아키텍처/디자인 제안)

- **Smart Cleanup**: 단순히 덮어쓰는 것이 아니라, 리소스(이미지)에 대한 참조 무결성을 수동으로 유지하여 서버 인프라 최적화.
- **Atomic Upsert**: 데이터베이스 엔진의 `UPSERT` 기능을 활용하여 경합 조건(Race Condition) 방지.
