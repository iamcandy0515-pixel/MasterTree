# Task: 수목 이미지 추출 상세 UI 개선 및 대표 이미지 로딩 수정

## 상태 기록 (Plan)

- **목적**: 상세 화면 진입 시 기존 이미지가 정상 노출되지 않는 버그를 수정하고, 사용자 요청에 따른 UI 레이아웃 재배치
- **분석 및 개선 포인트**:
    1. **AppBar 제목**: '가문비나무 이미지 관리' -> '수목 이미지 추출 상세'로 변경
    2. **헤더 레이아웃**:
        - '구글 드라이브 동기화' 버튼 우측 유지
        - 버튼 아랫줄에 현재 수목 이름(예: 가문비나무)을 별도로 표시하여 명시성 강화
    3. **이미지 박스 최적화**:
        - 현재 가로폭에 비해 세로가 너무 긴 박스 크기를 이미지 비율에 맞게 축소 (`childAspectRatio` 조정)
    4. **대표 이미지 미노출 버그**:
        - `Tree` 객체의 `images` 리스트에서 `imageType`이 `representative`인 이미지를 `main` 또는 `representative` 키로 정확히 `_stagedImages`에 로드하는지 점검
- **대상 파일**: `lib/features/trees/screens/tree_detail_screen.dart`
