# 작업 계획서: 문제/해설 섹션 이미지 UX 개선 및 붙여넣기 기능 통일 (관리자 앱)

## 1. 개요
관리자 앱의 '문제 상세 내용' 화면에서 이미지 편집 및 조회 UX를 대폭 개선합니다. 
1. **이미지 출력**: 썸네일 중심의 가로 스크롤 방식에서 벗어나, 사용자 앱처럼 지문/해설 텍스트 아래에 원본 비율의 대형 이미지를 즉시 노출합니다.
2. **붙여넣기(Ctrl+V) 일원화**: 현재 '해설' 섹션에만 노출되는 붙여넣기 안내 및 입력 영역을 '지문' 섹션에도 동일하게 적용하여 직관적인 이미지 추가가 가능하도록 합니다.

## 2. 보안 전략 및 준수 사항 (DEVELOPMENT_RULES.md)
- **로컬 Git 백업**: 작업 착수 전 및 중요 변경 시점에 `git commit` 수행
- **연쇄적 사고**: 사용자 앱의 검증된 레이아웃을 관리자 편집기에 최적화하여 이식
- **최소 수정**: 기존 `QuizReviewDetailScreen`의 `_buildSectionWithImages` 메서드를 고도화하여 중복 코드를 방지
- **검증**: 수정 후 `flutter run`을 통해 이미지 업로드, 삭제, 레이아웃 정렬 상태 확인
