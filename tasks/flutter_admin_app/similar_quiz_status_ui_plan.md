## 1. 문제 원인 분석
- **현상**: 문제 클릭 시 이미 저장된 유사 문제가 있음에도 "추천된 유사 문제가 없습니다"라고 표시됨.
- **원인**: `_showReviewDialog`가 AI 추천 결과가 담긴 메모리(`_tempRecommendations`)만 참조하고 있으며, DB에 저장된 `related_quiz_ids`를 기반으로 상세 정보를 페칭(fetch)해오는 로직이 부재함.
- **기능 부재**: 리스트 조회(`_fetchQuizzes`) 시 기존 완료 여부를 표시하지 않아 중복 작업 유발.
