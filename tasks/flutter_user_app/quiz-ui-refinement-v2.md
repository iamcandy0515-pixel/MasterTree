# Task: Quiz Screen UI & Blur Refinement

## 1. 상태 기록 (Plan)

- **목적**: '수목 식별 학습' 화면의 헤더 버튼 배치와 이미지 보안 블러 영역을 정교하게 조정하고, 다음 문제 버튼의 사용성을 개선함.
- **작업 범위**:
    1.  **헤더 버튼 재배치**: '그만하기' 버튼을 하단 섹션에서 상단 헤더의 '수목 식별 학습' 텍스트 옆(우측)으로 이동.
    2.  **이미지 블러 영역 조정**: 기존 1/4(0.25)에서 **1/8(0.125)** 영역으로 축소하여 핵심 이미지는 더 많이 보여주되 상/하단 보안은 유지.
    3.  **버튼 레이아웃 복구**: 하단 섹션의 '그만하기' 버튼을 제거하고, 정답 선택 후에만 나타나던 **'다음문제' 버튼**을 더 명확하게 노출하거나 배치를 최적화함.
