# Quiz Filter Validation

## 작업 목적

- '기출문제 연동' 시 사용된 검색필터(과목명, 년도, 회차)가 가져올 파일(pdf 추출) 내용과 논리적으로 일치하지 않을 때 즉시 차단하여, 불필요한 AI 토큰 소모와 잘못된 기출문제 DB 삽입을 방지

## 구체적 분석 및 계획 (Plan)

1. **프론트엔드(`flutter_admin_app`): 파일명(1차 필터) 고속 검증**
    - 선택된 Google Drive 파일명 (`fileNameText`)을 기준으로, 입력된 `Subject`, `Year`, `Round`를 문자 단위로 포함하고 있는지 우선 확인
    - `_initialYear`를 문자열 치환으로 제거한 후, `_initialRound` 번호가 파일명에 남아있는지 확인하여 "2013" 안의 "2" 인식 오류를 방지.
    - 과목명은 최소 앞 2글자 일치 확인 (예: `산림기사` 검색필터 시 파일명이 `산림필답_xxx`일 경우 '산림'으로 일치 인정)
    - 조건이 어긋날 시 `validateFilters(fileNameText)` 함수 내에서 예외 스로우하여 추출 전에 즉시 차단

2. **백엔드(`nodejs_admin_api`): 추가 안정장치(선택 사항이나 지금은 프론트에 먼저 집중)**
    - 기존 프롬프트 문자열이 `[]` 리턴을 종종 무시하는 현상 보완을 위해, 프론트엔드 단계에서 철저하게 거르는 것이 더 안전함.

## 결과

- `quiz_extraction_step2_viewmodel.dart` `extractQuiz` 부분에 유효성 검증 함수 `_validateBeforeExtract` 를 추가합니다.

## 잠재적 문제점 및 리스크 분석 (Risk Analysis)

- 과목명 검색 필터가 `조경산업기사` 처럼 다양할 때 파일명이 단순히 `조경산기` 로 되어있으면 앞 두 글자 `조경`으로 매칭됨. (적절함)
- 회차 번호가 한글 '일,이,삼' 등으로 되어 있으면 실패할 수 있으나 실무적으로 `1, 2, 3`을 사용 중이므로 문제없음.
