# 기출문제 연동 - 추출조건 과목명 및 UI 개선 (quiz-extraction-subject-ui-fix)

## 1. 목적 및 범위 (Plan)

- **목적:** '기출문제 연동' 화면 내 '추출조건'의 과목명 항목을 세분화(산림기사, 산림산업기사, 산업안전기사, 산업안전산업기사, 조경기사, 조경산업기사) 하고, 해당 조건 UI(Dropdown)를 테두리 없는 Text Combo Box 형태로 변경한다. 더불어 타이틀('문제') 옆에 직관적으로 과목명이 노출되도록 한다.
- **범위:**
    - `quiz_extraction_step2_viewmodel.dart`
    - `2_pdf_extraction_module.dart`
    - `3_question_explanation_module.dart`
    - `quiz_management_screen.dart` 등 추출 관련 힌트 텍스트 파일들

## 2. 작업 내용 (Execute)

- **추출조건 UI 보더 삭제:** `2_pdf_extraction_module.dart` 에서 각 조건(과목, 년도, 회차, 문제번호)을 감싸는 `Container` 의 `BoxDecoration` 및 `border` 속성을 제거하여, 텍스트 형태의 Border 없는 Dropdown 으로 UI를 간결하게 개선.
- **과목명 항목 리스트 업데이트:** 기존 '산림필답', '조경필답' 등의 그룹을 삭제하고 6가지 주요 자격증 (`산림기사`, `산림산업기사`, `산업안전기사`, `산업안전산업기사`, `조경기사`, `조경산업기사`) 목록으로 재구성.
- **파일명 기반 추천 자동 매핑 로직 수정:** `quiz_extraction_step2_viewmodel.dart` 의 `extractQuiz` 파싱 로직에서, 파일명에 포함된 단어에 따라 상세한 과목명(예: '산림산업기사' 등)이 매핑되도록 분기문 최적화.
- **제목('문제') 옆 과목명 변경:** `3_question_explanation_module.dart` 내 `vm.extractedSubject`를 가공 없이 그대로 출력하도록 변경. 기존에 '산림필답(기사, 산업기사)' 처럼 하드코딩된 예외 출력문을 제거.

## 3. 사후 점검 (Review)

- **Result (완료된 결과):** 추출조건 항목들이 깔끔한 텍스트 콤보 박스 형태로 표기되며, '산림산업기사' 등으로 정확히 나뉘어진 과목 리스트를 선택할 수 있다. '문제' 라벨 옆에도 선택/추출된 과목명이 그대로 직관적으로 연결되어 표기되며, 이 상태로 DB에도 성공적으로 저장됨.
- **Risk Analysis (향후 문제점 및 리스크 분석):** 기존 DB 내에 이미 '산림필답', '조경필답' 으로 저장된 데이터가 존재할 경우, 새롭게 구조화된 과목명(카테고리명)과 불일치할 수 있다. 필요하다면 데이터베이스의 기존 데이터를 마이그레이션(업데이트) 해주는 스크립트 실행이 요구됨.
