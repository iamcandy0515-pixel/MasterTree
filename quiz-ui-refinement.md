# Task: Quiz Screen UI Refinement - Hint Icons & Option Cards

## 1. 상태 기록 (Plan)

- **목적**: '수목 식별 학습' 화면의 시각적 요소들을 정교하게 조정하여 가독성과 심미성을 높임.
- **작업 범위**:
    1.  **힌트 아이콘 조정**: 아이콘 크기를 텍스트와 조화롭도록 축소하고 전체적인 레이아웃을 컴팩트하게 변경.
    2.  **보기 카드 개선**:
        - 기존 리스트 형태에서 독립된 '텍스트 카드' 형태로 변경.
        - 세로 간격을 넓히고 구분선(Divider)을 제거.
        - 카드 스타일(배경, 테두리, 그림자)을 적용하여 프리미엄 느낌 강화.

## 2. 실행 (Execute)

## 3. 사후 점검 (Review)

- **완료된 결과(Result)**:
    - **힌트 아이콘 최적화**: 텍스트 대비 너무 컸던 아이콘과 원형 배경의 크기를 줄여 시각적 밸런스를 맞추었습니다.
    - **보기 카드화**: 단순 목록 형태에서 고급스러운 카드 형태로 개편했습니다. 각 보기가 독립된 공간을 가지며, 터치 시 시각적 피드백(그림자 및 테두리 변화)이 강화되었습니다.
- **향후 문제점 및 리스크 분석(Risk Analysis)**:
    - **스크롤 길이**: 카드 간 간격(12px)과 내부 패딩 증가로 인해 한 화면에 보기가 다 들어오지 않을 수 있으나, 이미 `SingleChildScrollView`가 적용되어 있어 사용성에는 지장이 없습니다.
